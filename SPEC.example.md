## 3. 功能需求

### 3.1 核心功能 (P0)

| 功能 | 描述 | 验收标准 |
|------|------|----------|
| 用户认证 | 注册/登录/登出 | 支持邮箱密码登录，JWT token |
| 笔记创建 | 创建和编辑笔记 | 支持 Markdown，支持自动保存 |
| 笔记列表 | 查看所有笔记 | 支持按时间排序，显示摘要 |

### 3.2 重要功能 (P1)

| 功能 | 描述 | 验收标准 |
|------|------|----------|
| 笔记分类 | 标签和分类 | 支持多标签，标签筛选 |
| 搜索 | 全文搜索 | 支持标题和内容搜索 |
| 导出 | 导出为 PDF/Markdown | 支持单篇和批量导出 |

### 3.3 增强功能 (P2)

| 功能 | 描述 |
|------|------|
| 协作分享 | 分享笔记给他人 |
| 实时同步 | 多设备同步 |
| AI 辅助 | AI 写作建议 |

---

## 4. 非功能需求

### 4.1 性能

- 页面加载时间 < 2s
- 搜索响应时间 < 500ms

### 4.2 安全

- 所有数据传输使用 HTTPS
- 用户密码加密存储（bcrypt）
- JWT token 有效期 7 天

### 4.3 兼容性

- Chrome 90+
- Firefox 88+
- Safari 14+
- 移动端响应式布局

---

## 5. 技术约束

- 前端: React 18 + TypeScript
- 后端: Node.js + Express
- 数据库: PostgreSQL
